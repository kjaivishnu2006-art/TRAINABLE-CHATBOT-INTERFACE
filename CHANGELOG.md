## [0.1.0] - 2026-03-12

### Added
- Initial project setup for GSoC 2026
- Core training pipeline with TensorFlow/Keras
- Text preprocessing and tokenization
- Intent classification model
- Response template system
- Web-based training interface (Flask)
- Model export to TensorFlow Lite format
- Model quantization and optimization
- MIT App Inventor integration guide
- Comprehensive documentation
- Example chatbot templates (customer service, trivia)
- Unit and integration tests
- CLI interface for training and export
- GitHub Actions CI/CD pipeline
- Code quality checks (linting, type checking)

### Project Structure
- Organized source code (`src/` directory)
